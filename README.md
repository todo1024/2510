Ansible自动化运维应用实战
### 微:NoBug1024 


课程介绍：

ansible是新出现的自动化运维工具，基于Python开发，集合了众多运维工具（puppet、cfengine、chef、func、fabric）的优点，实现了批量系统配置、批量程序部署、批量运行命令等功能。无客户端。
